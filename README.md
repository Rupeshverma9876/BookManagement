## BOOK MANAGEMENT PROJECT
- A backend project designed to manage book records with full CRUD functionality and structured
architecture.
- Developed a complete Book Management System using Spring Boot and RESTful APIs for handling book records.
- Implemented end-to-end CRUD operations (Create, Read, Update, Delete) with real-time database connectivity using
MySQL and Spring Data JPA. 
  - Tested all API endpoints using Postman and ensured smooth data flow between controllers, services, and repositories.
  ## TECH STACK
  - JAVA
  - SPRING DATA JAP
  - MYSQL DATABASE
